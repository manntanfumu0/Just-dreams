# 🎮 Just Dreams

**Just Dreams** é uma plataforma gamificada de aprendizado que combina **quizzes interativos, acessibilidade, social e gamificação** para tornar o estudo mais divertido e inclusivo.  
O projeto inclui funcionalidades para **alunos, professores e administradores**, com foco em **inovação educacional**.
